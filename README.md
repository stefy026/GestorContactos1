# GestorContactos1
El Gestor de Contactos es una aplicación de consola desarrollada en C# que facilita la administración de información de contactos personales y profesionales. El sistema permite registrar datos completos (nombre, teléfono, email, dirección), realizar búsquedas avanzadas, categorizar contactos por tipo (Familia, Trabajo, Amigos) y generar estadísticas. Implementa validaciones de datos, persistencia en archivos y demuestra conceptos clave de POO como encapsulamiento, colecciones genéricas y LINQ.
